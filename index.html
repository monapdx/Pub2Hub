How It Works

  • Create a GitHub Repository

    • Use an existing repository or create a new one for your blog.

  • Create a Fine-Grained Token

    • Generate a GitHub fine-grained personal access token with read and write access to Actions, Contents, and Workflows.

  • Enable GitHub Pages

    • Open your repository settings and change the Pages source to GitHub Actions.

  • Connect Pub2Hub

    • Enter your GitHub username, repository name, and access token.

  • Start Publishing

    • Write your first post and click Publish. That's it.
